# Security Policy

## Supported Versions
mcpset is currently in early development (0.x). Always update to the latest release before reporting issues.

## Reporting a Vulnerability
1. Do **not** open a public issue.
2. Email security reports to `devdha+mcpset-security@gmail.com` with sufficient detail to reproduce the problem.
3. You will receive an acknowledgment within 72 hours. We'll coordinate a fix and public disclosure timeline together.

If you believe a vulnerability impacts other projects that depend on mcpset, please mention that in your report so we can coordinate responsibly.
